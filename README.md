## FAQ

**Q: What's Quran Tab?**  
A: Quran Tab is a browser extension that replaces your new tab with a random ayah from the Quran.  

**Q: How can I use it?**  
A: Download the extension from the Chrome Web Store or Edge Add-ons Store and it will replace your new tab with a random ayah.  

**Q: Is there a mobile app?**  
A: No, Quran Tab is currently only available as a browser extension.  

**Q: What about other Chromium browsers?**  
A: It works on all Chromium browsers that support replacing the new tab page like Brave and Vivaldi.
